Introducción

Con una inversión inicial razonable podemos adquirir una aplicación web que nos permita gestionar nuestro trabajo o actividad de forma mucho más eficiente.

Ventajas de las aplicaciones web

Hubo un tiempo, no hace mucho, en el que las aplicaciones de gestión "se instalaban" en los ordenadores. Pero esa tendencia cambió con el tiempo y hoy en día la mayoría de las aplicaciones de gestión son accesibles desde cualquier navegador web, como Microsoft Edge, Chrome o Firefox. Y ello representa varias ventajas:

  • Podemos acceder a nuestra aplicación desde cualquier lugar del planeta.
  • Usamos siempre la última versión de la aplicación.
  • Nuestra información personal está protegida frente malwares.

Primero móvil

Somos conscientes de que la mayoría de las personas acceden a Internet a través de dispositivos móviles, y ésa tendendcia se irá reforzando con el tiempo. Es por éllo que nuestra principal preocupación es desarrollar aplicaciones accesibles desde el propio móvil.

Proyectos

Gestor de fotografías online

Se trata de una aplicación para gestionar fotografías y adaptada a dispositivos móviles. La aplicación fue desarrollada en Angular, TypeScript, PHP y MySQL y usa los servicios de Dropbox como almacén de imágenes.

github.com/gchumillas/petitphotobox

Gestor de contenidos universal

Esta aplicación altamente configurable permite gestionar el contenido de cualquier sitio web, sin importar cómo haya sido desarrollado. La aplicación fue desarrollada primeramente en jQuery, PHP y MySQL.

cms.gchumillas.com/cms
